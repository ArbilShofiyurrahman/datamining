Pandas#

adalah modul yang digunakan untuk data analisis dan struktur data. Salah satu tools dari pandas adalah cut dan qcut.

cut menggunakan equal-width

Syntax cut#

pd.cut(series, interval, right=True, label)

,

sedangkan, qcut menggunakan equal-frequency.

Syntax qcut#

pd.qcut(series, interval, label)

Equal-width Intervals#

Equal-width intervals adalah discretization yang membagi data numerik menjadi beberapa kelompok dengan lebar kelompok yang kurang lebih sama besar.

labels = ["Sempit", "Lebar", "Sangat Lebar"]

# count every value based the given attribute
def countEveryAttribute(data_column, labels, column, attribute):
    group = data_column.groupby(attribute).count()
    amount_of_every_attribute = []
    for label in labels:
        amount_of_every_attribute.append(group.loc[label, column])
    return amount_of_every_attribute

# split the given data based on attribute and split value 
def split(split_value, data_column, labels, col, attribute):
    less_group = data_column[data_column[col] < split_value]
    greater_group = data_column[data_column[col] >= split_value]
    
    length_less_group = countEveryAttribute(less_group, labels, col, attribute)
    length_greater_group = countEveryAttribute(greater_group, labels, col, attribute)
    
    return (length_less_group, length_greater_group)

Fungsi menghitung perbedaan gain antara entropi inisial dengan entropi nilai pembagi yang diberikan#

Rumus Information Gain :

\[Gain(E_{ence}) = (E_{initial}) \ - (E_{ence})\]
# count information gain from inisial entropy and new entropy
def count_gain(inisial_entropy, new_entropy):
  return inisial_entropy - new_entropy

# count entropy
def count_entropy(data_target):
    all_prob = []
    for prob in data_target:
        if (prob/sum(data_target) != 0):
            all_prob.append(prob/sum(data_target) * log2(prob/sum(data_target)))
        else:
            all_prob.append(0)
    return -(sum(all_prob))

Fungsi menghitung Info entropy berdasarkan nilai pembagi atau split value yang diberikan#

\[ Info_A(D) = \frac{|D_1|}{|D|} Entropy (D_1)\frac{|D_2|}{|D|}+ Entropy (D_2)\]
# count entropy for given a split value
def info(d, data_target):
    temp = []
    for value in d:
        temp.append((sum(value) / data_target.shape[0]) * count_entropy(value))
    return sum(temp)
